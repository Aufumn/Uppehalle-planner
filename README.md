# -Uppehalle-planner
Research project for COMP-SCI 580, the plan is to make a program that can reccomend meals to people who have allergies to food.

## Project overview
My project is a webpage that will allow you to put in your dietary restrictions and git recipies back from a database that match your restricitons. the goal of this tool is to allow people to find safe foods to eat that can also allow them to try new things.
